# Balle Bot — Landing Page

Landing page de apresentação do **Balle Bot**, um bot de moderação para comunidades online. Desenvolvida com HTML e CSS puro como meu primeiro projeto no GitHub, enquanto eu aprendia programação.

## Sobre o projeto

A página apresenta o Balle Bot e seus três recursos principais:

- **Captcha de verificação** — novos membros provam que não são robôs
- **Banimento automático** — links suspeitos são removidos automaticamente
- **Sistema de avisos e banimento manual** — ferramentas para administradores

## Tecnologias

- HTML5
- CSS3 (Flexbox, gradientes, Google Fonts)

## Como visualizar

Basta abrir o arquivo [index.html](index.html) no navegador — nenhuma dependência ou build necessário.
